RA-Crew

A multi-agent AI system for automated extraction, validation, and analysis of SEC EDGAR filings (10-K, 10-Q, 8-K, DEF 14A). Built with CrewAI, LangChain, Retrieval-Augmented Generation, and robust validation layers for research-grade financial data.

Architecture Overview

DataRetriever

Authenticated, rate-limited SEC EDGAR access with ticker-to-CIK resolution.

  • CIK normalization and caching
  • Retry + exponential backoff
  • Multi-year filing traversal
  • HTML / XML / Text acquisition

DataCleaner

Preserves semantic structure while removing noise for downstream extraction.

  • HTML parsing & tag pruning
  • Table boundary preservation
  • Unicode & whitespace normalization
  • Chunk-safe formatting

DataExtractor

Hybrid RAG + pattern methods for executive compensation and metric mining.

  • Query-driven relevance search
  • Compensation table parsing
  • Regex + LLM hybrid scoring
  • Context traceability

DataCalculator

Deterministic financial computations executed in sandboxed Python.

  • Unit-aware math ops
  • Safe evaluation wrapper
  • Derived metric synthesis
  • Error surface transparency

DataValidator

Cross-field & longitudinal validation for integrity and plausibility.

  • Outlier detection
  • Null policy enforcement
  • Monetary unit checks
  • Consistency summaries

GraduateAssistant

Coordinator agent orchestrating workflows, retries, and batching.

  • Adaptive delegation
  • Error recovery logic
  • Throughput optimization
  • Progress multiplexing

Quickstart

# Environment
conda create -n ra-crew python=3.12
conda activate ra-crew
pip install -r requirements.txt

# Configure
cp .env.example .env
# edit .env with keys + EDGAR_IDENTITY

# Interactive run
python -m ra_crew.cli --interactive

# Direct batch
python -m ra_crew.cli \
  --companies "AAPL:2023,MSFT:2022" \
  --filings "DEF 14A" \
  --metrics "Total CEO compensation,Total CFO compensation" \
  --use-crew --verbose

Design Principles

  • Determinism First: Calculations executed outside LLM reasoning
  • Traceability: Every extracted value links to contextual source
  • Resilience: Retry, rate control, and defensive parsing
  • Extensibility: Pluggable providers & metric templates
  • Human Review: Structured JSON artifacts enable auditing
  • Anti-Hallucination: Null instead of fabricated placeholders
